Experimental!

Unlimited Image Combiner는 빠르게 여러 이미지를 하나의 이미지로 합치는 실험적인 웹 앱입니다. 인터넷 익스플로러 10과 구글 크롬, 모질라 파이어폭스의 최신 버전에서 동작합니다.

우측 상단의 '파일 선택' 버튼을 눌러 이미지를 선택한 하세요. 화면에 미리보기가 표시되면 이미지가 모두 합쳐진 것입니다. 이 때 '이미지 저장' 버튼을 눌러 다운로드하세요.

이미지의 개수에 제한이 없으나, 높이가 16000px을 초과하게 되면 한 개 이상의 이미지로 분할되어 합쳐집니다. 많은 이미지를 합친 후 다운로드 할 때에는 시간이 걸릴 수 있으니 기다려주세요.
